## 什么是搬瓦工的 KiwiVM 面板

搬瓦工用的是自研控制面板，叫 KiwiVM Control Panel。简单说，它是你 VPS 的"遥控器"——开关机、重装系统、查流量、换机房、改密码，全在里面。**搬瓦工登录 VPS 后的第一步**，几乎都从这个面板开始。

不用安装，浏览器打开就能用。进入路径：登录搬瓦工账号 → 顶部菜单 Services → My Services → 找到你的 VPS → 点击右侧「KiwiVM Control Panel」按钮。

## 第一步：找到 IP 地址、SSH 端口和 root 密码

三样东西，缺一个都连不上：

- **Public IP address**：服务器的外网 IP
- **SSH Port**：不是默认 22，搬瓦工用的是随机高位端口，比如 28785
- **root 密码**：重装系统时随机生成，只出现一次

KiwiVM 面板首页「Main controls」区域可以直接看到 IP 和 SSH 端口。密码的话——如果当时没记下来，就在面板里找「Root password modification」，先把 VPS 关机，然后重置，会生成一个新密码。记住，关机是前提，开着机重置会失败。

## 第二步：用 SSH 客户端连进去

Windows 上用 Termius 或者 Xshell，两个都有免费版。Mac 直接开终端就行，不用装额外工具。

连接命令格式：

bash
ssh root@你的IP地址 -p SSH端口号


比如 IP 是 `1.2.3.4`，端口是 `28785`，就是：

bash
ssh root@1.2.3.4 -p 28785


输完回车，提示输密码，注意这时候屏幕不会显示你在打字，这是正常的，输完直接回车就好。

出现 `root@localhost:~#` 之类的提示符，说明进去了。

## 搬瓦工登录 VPS 后：标准初始化流程

进去之后先别急着装东西。按这个顺序做，后面省事：

**1. 先更新系统**

根据你装的系统选命令：

- Ubuntu/Debian：`apt update && apt upgrade -y`
- CentOS：`yum update -y`

搬瓦工新买的 VPS 默认装的是 CentOS，不过现在很多人会重装成 Ubuntu 22.04 或 24.04，社区教程多，出了问题好搜。

**2. 改 SSH 端口（可选但推荐）**

默认端口或者当前的随机端口，可以改一个你自己记得住的。编辑 `/etc/ssh/sshd_config`，找到 `Port` 那行改掉，然后 `systemctl restart sshd`。改完之前别关掉当前会话，先开新窗口测试连接成功再退出。

**3. 设置防火墙**

Ubuntu 上直接用 `ufw`：

bash
ufw allow 你的新SSH端口
ufw enable


CentOS 上用 `firewalld` 或者 `iptables`。

**4. 创建普通用户，不要一直用 root**

长期用 root 是不好的习惯，新建一个用户，用 `sudo` 来执行特权命令：

bash
adduser 用户名
usermod -aG sudo 用户名


做完这四步，服务器的基础状态就算稳了，接下来才是按需求装东西。

## KiwiVM 面板的几个常用功能

**进去之后不知道各个按钮是干什么的？** 简单说几个高频用到的：

| 功能名称 | 在哪里 | 干什么 |
|---|---|---|
| Main controls | 首页 | 开关机、重启、查 IP 和端口 |
| Install new OS | 左侧菜单 | 重装系统（关机后才能操作） |
| Root password modification | 左侧菜单 | 重置 root 密码 |
| Migrate to another DC | 左侧菜单 | 换机房（会换 IP） |

网页版 SSH 就是 KiwiVM 自带的，SSH 端口被封、或者客户端连不上的时候，打开这个还能操作。用户名 root，密码跟 SSH 一样。

## 搬瓦工登录后常见问题

**Q：买完之后没收到 SSH 信息怎么办？**

直接去 KiwiVM 面板看就行。Services → My Services → 点开 KiwiVM，首页有 IP 和端口。密码重置也在面板里操作。

**Q：SSH 连不上，提示 Connection refused？**

几种可能：VPS 没开机（去面板 Start）；端口填错了（确认一下 KiwiVM 里显示的端口）；IP 被封了（用 KiwiVM 自带 IP 检测工具查一下，或者直接换机房）。

**Q：搬瓦工登录 VPS 后流量快用完了怎么办？**

KiwiVM 面板首页能看到当月流量使用情况。流量用完 VPS 会暂停，有个「临时恢复」功能可以临时解封继续使用，但流量用完就停。下个月 1 号会重置。

**Q：重装系统会丢数据吗？**

会。重装前先打快照。快照在 Snapshots 里建，免费 2 个，30 天有效。

**Q：搬瓦工支持退款吗？**

30 天内不满意可以申请全额退款。在账号后台提交工单走退款流程，有用过的朋友实测退回来了，没有问题。
